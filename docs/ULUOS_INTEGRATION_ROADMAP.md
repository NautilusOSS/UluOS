# UluOS Integration Roadmap
Version: v0
Status: Draft

## Overview

UluOS is a composed runtime formed by the Ulu MCP ecosystem behind a unified gateway control plane.

## Layers

- Infrastructure: UluCoreMCP, UluWalletMCP, UluBroadcastMCP
- Meaning: UluVoiMCP, UluAlgorandMCP
- Protocol: DorkFiMCP, EnvoiMCP
- Control Plane: UluGateway

1. Assembly
2. Developer Alpha
3. Developer Experience
